DESCRIPTION OF 3RD STATISTIC:

Instead of having point values, the places that decathletes placed in for each event are
displayed. Ff athletes did not finish, or scores are not avaliable for the event,
it is replaced with a DNF. Ordered by highest total points to make it look cleaner.
